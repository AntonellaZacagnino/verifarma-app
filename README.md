## Welcome to VeriMovies!

## Setup  

Before you clone this project, you need to install the dependencies

```bash
# npm
npm install

# yarn
yarn install
```

Then, run the project on development server

```bash
#npm
npm run dev

#yarn
yarn dev
```

The project will run on http://localhost:3000/

## Developed with

* Vue.js
* Nuxt.js
* NuxtUI
* Firebase
* SASS
